# 自制操作系统

制造自己的操作系统，补足知识体系。

# 计划

1. 参考已有的实现，了解制作操心系统需要哪些知识，补足自己缺少的知识。

2. 列出完成目标的大纲。

> 通过已有的实现，了解到自己缺少`汇编语言`和`80386 CPU`相关的知识。

所以，将计划大致分为三个阶段：

1. [汇编语言学习](./gas/index.md)
2. 80386 CPU 学习
3. 自制操作系统

# 环境

qemu、windows wsl、GNU Binutils、gcc
